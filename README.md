# 绝区零伤害公式计算器

一个用于展示和查询绝区零游戏中各种伤害计算公式的在线工具。

## 📋 项目简介

本项目提供了绝区零游戏中的详细伤害计算公式，包括通用伤害、防御区、直伤、命破、异常伤害、紊乱等核心战斗机制的数学公式，帮助玩家深入理解游戏的伤害计算系统。

直接用浏览器打开 `绝区零伤害公式.html` 即可使用，无需额外配置。

4. **查看详情**：每个公式都配有详细的参数说明和计算逻辑

## 📊 公式说明

### 伤害计算核心

#### 通用伤害

基础伤害计算框架，包含：

- 基础属性 × 倍率
- 失衡易伤、增伤、暴击伤害
- 抗性计算（增抗、减抗、抗穿）

#### 防御区

敌人防御力对伤害的影响：

- 等级基础防御
- 加防、减防、无视防御
- 穿透值和穿透百分比

### 伤害类型

#### 直伤伤害

标准攻击伤害，适用于大部分技能：

- 基于攻击力
- 受防御区影响
- 易伤、减伤、距离衰减

#### 命破伤害

特殊的贯穿伤害机制：

- 贯穿力 = 生命值/10 + 3×攻击力/10
- 不受距离衰减影响
- 独立的贯穿增伤系统

### 异常系统

#### 异常伤害

五种异常状态的持续伤害：

- **强击**：713% × 1次
- **碎冰**：500% × 1次
- **灼烧**：50% × 20次（0.5秒CD）
- **感电**：125% × 10次（1秒CD）
- **侵蚀**：62.5% × 20次（0.5秒CD）

#### 异常积蓄

触发异常状态的积蓄机制：

- 基础积蓄值 × 异常掌控
- 积蓄效率提升/降低
- 积蓄抗性影响
- 阈值递增机制（1.05倍）

#### 紊乱反应

两种异常状态碰撞产生的反应：

- 基础倍率：450%
- 额外倍率：根据剩余异常时间计算
- 失衡倍率：200%

### 特殊机制

#### 加权规则

异常伤害和积蓄的贡献分配：

- 按攻击贡献占比参与结算
- 确保多人/多次攻击的公平性

## 🔄 数据维护

### 更新流程

1. 在 `绝区零伤害公式.md` 中更新或添加公式
2. 同步更新到HTML文件中
3. 保持完整版和拆分版数据一致

### 数据格式

数学公式使用 LaTeX 语法，通过 MathJax 渲染：

```latex
$$
\text{伤害} = \text{基础属性} \times \text{倍率} \times (1 + \text{增伤}\%)
$$
```

### 版本同步

- 完整版和拆分版的公式保持同步
- 更新时需同时维护两个版本
- 建议使用脚本自动同步
