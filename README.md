# 绝区零计算器工具集

绝区零游戏伤害计算公式查询及代理人配装毕业计算工具。

## ✨ 主要功能

### 伤害公式查询器
- **通用伤害**：基础属性、倍率、增伤、暴伤、抗性计算
- **防御区**：防御力、减防、无视防御、穿透机制
- **直伤伤害**：攻击力基础的标准伤害
- **命破伤害**：贯穿力系统（生命值+攻击力）
- **异常伤害**：强击、碎冰、灼烧、感电、侵蚀五大异常状态
- **异常积蓄**：异常掌控、积蓄效率、阈值递增
- **紊乱反应**：异常碰撞产生的额外伤害
- **数学公式渲染**：使用 MathJax 完美展示复杂公式

### 毕业计算器（NEW 🎯）
- **智能配装**：自动寻找最优驱动盘组合与副词条分配
- **属性规划**：输入基础属性和目标值，计算最优方案
- **概率分析**：显示每个配装方案的命中概率
- **数据导出**：支持导出所有可行组合方案为TXT文件
- **多属性支持**：生命值、攻击力、暴击率三维属性计算

### 界面特性
- 🌓 **深浅色模式**：支持主题切换，保护视力
- 📱 **响应式设计**：完美适配桌面和移动端
- 🎨 **现代化UI**：卡片式布局，清晰的视觉层次
- ⚡ **流畅体验**：弹窗式设计，无需页面跳转

## 🚀 使用方法

### 完整版
直接打开 `绝区零伤害公式.html` 即可使用。

### 模块化版本
打开 `绝区零伤害公式拆分/index.html`，方便自定义修改样式和功能。

## 📊 毕业计算器使用指南

### 基础操作
1. 点击顶部 "🎯 毕业计算器" 按钮打开弹窗
2. 输入代理人名称和武器档位
3. 设置基础属性和目标属性（生命值、攻击力、暴击率）
4. 选择驱动盘主词条和套装（可选"自动"）
5. 点击"自动毕业计算"或"使用当前选择重新计算"

### 计算结果
- **概率总和**：当前配装方案的整体命中概率
- **最优方案**：自动推荐概率最高的配装组合
- **盘位详情**：每个盘位的副词条需求和单独概率
- **导出功能**：可导出所有可行组合方案供深入分析

### 示例场景
```
代理人：朱鸢
武器档位：713
基础攻击力：800 → 目标攻击力：2500
基础暴击率：5% → 目标暴击率：70%

→ 系统自动计算：
  - 最优主词条组合
  - 最优套装搭配
  - 所需副词条次数
  - 毕业概率
```

## 📋 伤害公式详情

### 核心伤害类型
- **直伤**：攻击力 × 倍率 × 防御区 × 易伤区
- **命破**：贯穿力 × 倍率 × 易伤区 × 贯穿增伤区

### 异常伤害倍率
| 异常类型 | 倍率 | 次数 | 频率 |
|---------|------|-----|------|
| 强击 | 713% | 1次 | - |
| 碎冰 | 500% | 1次 | - |
| 灼烧 | 50% | 20次 | 0.5s/次 |
| 感电 | 125% | 10次 | 1s CD |
| 侵蚀 | 62.5% | 20次 | 0.5s CD |

### 异常积蓄阈值
- 普通敌人：600（物理720）
- 精英敌人：2250（物理2700）
- 首领敌人：3000（物理3600）
- 每次触发后阈值 ×1.05
